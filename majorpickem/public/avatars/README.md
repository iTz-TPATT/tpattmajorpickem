# Player Avatars

Upload headshots here. Filename = the player's login name, lowercase, spaces replaced with hyphens.

## Examples
| Login Name | Filename |
|---|---|
| Trenton Patterson | trenton-patterson.jpg |
| Adam Gourley | adam-gourley.jpg |
| Will Robson | will-robson.jpg |
| John Smith | john-smith.jpg |

## Rules
- JPG or PNG both work
- Square crops look best (400x400 minimum)
- If no photo is uploaded, a colored initial badge shows instead
- Filenames are case-insensitive and ignore extra spaces
