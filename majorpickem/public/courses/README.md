# Course Photos

Add your golf course photos here. Filenames must match exactly:

- masters2.jpg — Amen Corner or 11th/12th/13th
- masters3.jpg — 16th hole Redbud or aerial
- masters4.jpg — Any Augusta landscape

## The Open (Royal Portrush)

## Tips
- Landscape orientation works best (wider than tall)
- Aim for 1400px wide or larger for best quality
- JPG format preferred
- If a file is missing, the app automatically shows a placeholder photo
